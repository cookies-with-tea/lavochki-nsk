# Benches Backend

# Установка проекта

- скопировать репозиторий
    ```console
    $ git clone https://github.com/anime-team-pet-projects/lavochki-nsk.git -b backend/production benches-backend
    $ cd benches-backend
    ```
- установить `.env` файл
- запустить контейнер
    ```console
    $ docker-compose up --build -d
    ```
